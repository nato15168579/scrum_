───────────────────────────────────────────────────────────────────────────────
🔍 VERIFICAR BASE DE DATOS (IMPORTANTE)
───────────────────────────────────────────────────────────────────────────────

Ejecutar en MySQL:

  SELECT usuCedula, usuNombres, usuContrasena, rolSisIdFk 
  FROM usuario LIMIT 1;

Requisitos:
  ✅ usuContrasena comienza con $2b$ (bcrypt hasheada)
  ✅ rolSisIdFk tiene un valor (2=Instructor, otro=Aprendiz)

Si no cumple:
  UPDATE usuario SET rolSisIdFk = 2 WHERE usuCedula = 1234567890;

🧪 PRUEBA DE LOGIN
───────────────────────────────────────────────────────────────────────────────

1. Abrir: http://localhost:5173
2. Presionar: F12 (DevTools)
3. Ir a: Console tab
4. Ingresar cédula del usuario y contraseña

Debería ver en Console:
  ✅ Iniciando login con cédula: ...
  ✅ Datos recibidos del servidor: {...}
  ✅ Rol es Instructor (2) - Redirigiendo a /dashboard-instructor
  O
  ✅ Rol es Aprendiz - Redirigiendo a /student-dashboard

Si no aparecen ✅:
  Revisar VERIFICACION_LOGIN.md → Troubleshooting
